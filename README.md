# Школа оклейки автомобилей Екатеринбург

Это Next.js приложение для сайта школы оклейки автомобилей в Екатеринбурге.

## Описание

Сайт включает разделы:
- Герой секция
- О школе
- Преподаватели
- Кому подойдет обучение
- Как проходит обучение
- Что вы получите
- Программа обучения
- Пакеты обучения
- Отзывы учеников
- Партнеры
- Контакты

## Запуск

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите сервер разработки:
   ```bash
   npm run dev
   ```

3. Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Сборка

```bash
npm run build
```

## Развертывание

Проект можно развернуть на Vercel, Netlify или другом хостинге для Next.js.
